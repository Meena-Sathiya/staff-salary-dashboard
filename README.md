Staff Salary Dashboard using Excel and Power BI
Staff Salary Dashboard using Excel and Power BI

This project focuses on analysing employee salary data and building an interactive dashboard.

Key tasks performed:

• Cleaned and transformed raw dataset using Power Query
• Handled missing values and corrected data types
• Analysed salary distribution by department, work mode, and location
• Built interactive Power BI dashboard with filters and visuals

Tools used:
Excel, Power Query, Power BI
